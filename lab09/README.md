# Система массового обслуживания M/M/1

Система массового обслуживания M/M/1 - одноканальная система без очереди: если сервер занят в момент прихода заявки, она отклоняется. Система обладает следующими свойствами:

- **Входной поток** - пуассоновский с интенсивностью λ: интервалы между заявками распределены по показательному закону τ ~ Exp(λ).
- **Время обслуживания** - показательное с параметром μ: s ~ Exp(μ).
- **Один канал** - сервер либо свободен (состояние 0), либо занят (состояние 1).

## Теоретические характеристики

Нагрузка системы:

$$\rho = \frac{\lambda}{\mu}$$

Стационарное распределение:

$$P_0 = \frac{1}{1 + \rho}, \quad P_1 = \frac{\rho}{1 + \rho}$$

где P0 - вероятность того, что сервер свободен, P1 - вероятность потери заявки (сервер занят).

**Пропускная способность:**

$$A = \lambda \cdot P_0$$

### Алгоритм моделирования

1. Генерируется N интервалов между заявками: τ_i ~ Exp(λ), времена прихода t_i = Σ τ_j.
2. Генерируется N времён обслуживания: s_i ~ Exp(μ).
3. Для каждой заявки:
   - если t_i ≥ t_свободен - заявка принята, t_свободен = t_i + s_i;
   - иначе - заявка **выброшена**, t_свободен не изменяется.
4. Эмпирические вероятности:

$$P_0^{\text{эмп}} = \frac{\text{принятых}}{N}, \quad P_1^{\text{эмп}} = \frac{\text{потерянных}}{N}$$
