### Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

### Описание методов

**MCG (Мультипликативный конгруэнтный генератор)**  
Формула: `Xₙ₊₁ = (β · Xₙ) mod M`  
Параметры: `β = 2³² + 3 = 4 294 967 299`, `M = 2⁶³ = 9 223 372 036 854 775 808`

**Метод Фибоначчи (Аддитивный генератор)**  
Формула: `Xₙ = (Xₙ₋₁ + Xₙ₋₂) mod M`  
Параметры: `M = 2⁶³`, начальные значения генерируются из seed

**Встроенный генератор (NumPy)**
используется как эталон качества.

**Теоретические значения**  
- Среднее: 0.5  
- Дисперсия: 1/12 ≈ 0.08333

### Результаты (N = 100 000, seed = 42)

| Метрика | MCG | Фибоначчи | Встроенный | Теория |
|---------|-----|-----------|-----------|--------|
| Среднее | 0.50219 | 0.49724 | 0.49416 | 0.50000 |
| Дисперсия | 0.08411 | 0.08570 | 0.08272 | 0.08333 |

### Выводы
1. Все три генератора показывают хорошее соответствие теоретическому равномерному распределению на интервале [0, 1):
   - Отклонения среднего от теоретического значения (0.5) не превышают 0.006
   - Отклонения дисперсии от теоретического значения (0.08333) не превышают 0.0024

2. **MCG** с параметрами со слайда (β = 2³² + 3, M = 2⁶³) показал наилучшее приближение к теоретическому среднему (0.50219), что свидетельствует о хорошем качестве генератора.

3. **Метод Фибоначчи** продемонстрировал несколько большее отклонение по дисперсии (0.08570), что может быть связано с корреляцией между последовательными значениями, характерной для аддитивных генераторов.

4. **Встроенный генератор NumPy** показал результаты, близкие к теории, с небольшим смещением среднего в меньшую сторону (0.49416).
