### Клеточные автоматы. Лесные пожары (GUI)

**Задание:**  
Реализовать моделирование возникновения и распространения лесных пожаров с использованием двумерного клеточного автомата.

**Требования:**
- реализовать **не менее трёх дополнительных правил** поведения системы.


## Состояния клеток

Каждая клетка может находиться в одном из следующих состояний:

| Состояние | Описание |
|---|---|
| EMPTY | пустая клетка |
| TREE | клетка с деревом |
| BURNING | дерево горит |
| WATER | клетка с водой |

Состояние WATER является барьером для распространения пожара.

## Окрестность клетки

Для взаимодействия клеток используется **окрестность Мура**, включающая соседние клетки, окружающие рассматриваемую клетку. Каждая клетка может взаимодействовать максимум с восьмью соседями.

## Типы деревьев

В программе есть три типа дерева, у каждого из которых различное время горения (шаги горения) и коэффициент воспламеняемости.

| Тип дерева | Время горения | Коэффициент воспламеняемости |
|---|---|---|
| 1 | 3 | 0.7 |
| 2 | 2 | 0.9 |
| 3 | 5 | 0.4 |

## Рост деревьев

На пустых клетках может происходить рост новых деревьев. Вероятность появления дерева определяется параметром, вводимым пользователем.

Дождь оказывает положительное влияние на рост растительности. В условиях дождя вероятность роста увеличивается на 80%.

## Самопроизвольное возникновение пожара

Деревья могут воспламеняться без непосредственного контакта с огнём. Вероятность самовозгорания определяется параметром, который также задается пользователем.

## Распространение огня

Основной механизм развития пожара связан с передачей огня от горящих клеток к соседним деревьям. Вероятность воспламенения дерева зависит от двух факторов:
- базовой вероятности распространения огня;
- коэффициента воспламеняемости конкретного типа дерева.

## Влияние ветра

Ветер влияет на направление распространения огня. Направление и сила ветра задаётся с помощью "компаса", 

Влияние ветра на распространение огня определяется следующим образом:
- при совпадении направления ветра с направлением распространения огня вероятность возгорания увеличивается;
- при противоположном направлении вероятность уменьшается.

## Влияние дождя

Дождь оказывает комплексное влияние на систему. Во-первых, он снижает вероятность распространения огня. В условиях дождя вероятность передачи огня уменьшается на 70%. Во-вторых, дождь может приводить к тушению уже горящих деревьев.

## Образование луж

Во время дождя на пустых клетках могут образовываться временные водные участки — лужи. Появление луж увеличивает долю клеток воды на карте.

## Водные преграды

Клетки, находящиеся в состоянии WATER, выполняют роль естественных барьеров. Через такие клетки огонь не распространяется, поскольку вода не может воспламеняться. 

## Динамика горения

Горящее дерево не исчезает мгновенно. Для каждого типа дерева задаётся время горения. На каждом шаге моделирования значение времени горения уменьшается. После завершения процесса горения клетка становится пустой.

## Начальная генерация карты

Перед началом моделирования формируется начальное состояние карты. Генерация включает несколько этапов:
1. создание водных объектов (реки и озёра);
2. заполнение свободных клеток деревьями с заданной плотностью леса;
3. назначение каждому дереву случайного типа.
